# How to contribute

## Run tests

``` sh
# Bundle gems
bundle

# Install npm packages
npm install

# Run tests
bundle exec guard
```

## Sending pull-request

1. Fork,
2. run the tests,
3. add test for your change,
4. make tests green,
5. push,
6. send pull-request.

## Please do not

* Driving drunk,
* hide the fact of finding Bigfoot,
* mess with versions, gemspec file, package.json etc.
